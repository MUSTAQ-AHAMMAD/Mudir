# 🎨 Mudir — Visual Design Prompts

Copy these into an AI image generator (DALL·E 3, Midjourney, Stable Diffusion,
Canva AI, or Figma AI plugins) to produce the app's visual assets. Brand colors:
**deep green `#0d5c36`**, **green `#1a7a4a`**, **gold `#d4af37` / `#c9a84c`**.

---

## Visual 1 — WhatsApp Chat Interface (mockup)
> Create a realistic mobile phone screenshot showing a WhatsApp group chat called
> "Riyadh Mall Store Opening." The chat shows 3 team members (Property, Marketing,
> IT) and the AI bot "Mudir" responding. Show the bot's messages with: ✅ green
> checkmarks for completed tasks; ⚠️ yellow warnings for overdue tasks; 📋 a blue
> message with a task assignment. The bot has a distinct profile picture (a
> crescent moon and gear icon). The bottom shows the WhatsApp text input with "/"
> suggesting commands.

## Visual 2 — Admin Dashboard
> Create a modern, minimal web dashboard for Saudi Arabian retail companies.
> Left sidebar navigation (Dashboard, Projects, Teams, Analytics, Settings). Main
> content shows a project timeline with 3 columns (Property, Marketing, IT); each
> column has cards showing task status (Pending, In Progress, Complete). A green
> "Store Opens On Time" banner at the top. Color scheme: dark green (#1a7a4a),
> gold (#c9a84c), white background. Mixed Arabic/English text (e.g. "المشاريع /
> Projects"). Responsive for desktop and tablet.

## Visual 3 — Project Timeline (Gantt)
> Create a horizontal Gantt-chart timeline for a store opening project. 15-day
> timeline (Day 1–15). Three colored bars: Property (blue), Marketing (orange),
> IT (purple). Dependency arrows Property → Marketing → IT. A red flag marker
> where Marketing was delayed by 2 days. A green checkmark at Day 15 showing
> "Store Opened." Clean infographic style for a business presentation.

## Visual 4 — Mobile App (future version)
> Design a mobile app interface for project coordinators. Home screen with
> "Active Projects" cards; each card shows project name, current team, days
> remaining. A bell notification icon with a red badge for escalations. Bottom
> navigation: Projects, Teams, Messages, Profile. Saudi-inspired design elements
> (geometric patterns, desert colors). Dark mode with gold accents.

## Visual 5 — Bot Interaction Flow
> Create an infographic showing the flow of a store opening project as a series
> of WhatsApp bubbles: (1) Property types "/new_project"; (2) Bot confirms;
> (3) Property completes → Bot notifies Marketing; (4) Marketing completes → Bot
> notifies IT; (5) IT completes → Bot runs final checklist; (6) Bot announces
> "Store Ready!". Arrows connect each step. Modern flat design, colorful but
> professional.

## Visual 6 — Logo
> Design a professional logo for "Mudir" (Arabic for Manager): a stylized crescent
> moon combined with a gear/settings icon. Colors deep green (#0d5c36) and gold
> (#d4af37). Arabic calligraphy of "مدير" below the icon and English "Mudir" in a
> modern sans-serif font. Minimalist, tech-forward, suitable for a square app icon.
